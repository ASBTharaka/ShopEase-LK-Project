# ShopEase LK — Business Analysis Project

## Overview
ShopEase LK is a proposed Sri Lankan e-commerce platform designed to improve trust, delivery flexibility, and cash payment support for online shopping.

This project was created as a Business Analysis portfolio project to demonstrate:

- Requirement Gathering
- Business Process Understanding
- BRD & SRS Documentation
- Stakeholder Analysis
- User Story Creation
- Wireframing
- Functional Requirement Definition

---

## Project Documents

| Document | Description |
|---|---|
| Project Scope Statement | Defines project objectives, scope, risks, and success criteria |
| Stakeholder Register | Identifies stakeholders and responsibilities |
| BRD | Business requirements and business needs |
| SRS | Functional and non-functional requirements |
| User Stories | Agile user stories and acceptance criteria |
| Wireframes | UI/UX low-fidelity screens |

---

## Key Features

- Grocery & general product ordering
- Seller verification system
- Cash on Delivery support
- Pickup-point delivery model
- PickMe/Uber urgent delivery integration
- Real-time order tracking
